HTML이란 무엇인가

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are tlohe building bcks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.

태그: TAG

  1. strong : 진하게
  2. u : 밑줄
  3. h1 ~ h6 : 제목
  4. br : 줄바꿈 (닫는 태그 필요 X)
  5. p : 단락
  6. img : 이미지
  7. li : 목차, 목록
  8. ul : li의 부모 태그(unordered list)
  9. ol : 목차에 자동 넘버링 (ordered list)
  10. a : 링크(anchor)
